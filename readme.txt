== Description ==

= In English =

Author : [Burak Sirri Vanli](http://www.burakvanli.com "Burak Sirri Vanli Personal Blog")

Special contextual advertising management on your blog by using links to assigned keyword terms in post content. English and Turkish language support.


* Only visual contextual ad.
* Only text contextual ad.
* Visual and text contextual ad.
* Standard link ad.

Visit the [plugin website.](http://www.contextualpress.com "Contextualpress web site")

= In Turkish =

Turkce icin eklentinin [web sitesini](http://www.contextualpress.com/tr/ "Contextualpress") ziyaret edin.

== Installation ==

In most cases you can install automatically from WordPress.

However, if you install this manually, follow these steps:

1. Upload the entire contextualpress folder to the '/wp-content/plugins/' directory.
1. Activate the plugin through the 'Plugins' menu in WordPress
